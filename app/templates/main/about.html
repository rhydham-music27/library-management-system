About Our Library

This Library Management System helps track books, members, and borrowing activity. It is built with a modern Flask architecture using SQLAlchemy and Bootstrap 5.

System Information

  • Version: 0.1.0 (foundation)
  • Framework: Flask
  • Database: SQLite (WAL mode)
  • ORM: SQLAlchemy
  • UI: Bootstrap 5
